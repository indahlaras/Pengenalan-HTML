<!DOCTYPE html>: Deklarasi wajib untuk memberitahu browser bahwa dokumen ini menggunakan standar HTML5.
<html>: Akar dari seluruh dokumen. Semua kode HTML harus ada di dalam sini.
<head>: Wadah untuk informasi "di balik layar" yang tidak muncul langsung di badan halaman.
<meta charset="UTF-8">: Menentukan pengkodean karakter agar simbol dan huruf dari seluruh bahasa dapat terbaca dengan benar.
<meta name="viewport" ...>: Mengatur agar tampilan website menyesuaikan lebar layar perangkat (penting agar web mobile-friendly).
<title>: Memberikan judul pada tab browser. Sangat penting untuk identitas web.
<meta name="description" ...>: Ringkasan singkat isi web untuk mesin pencari seperti Google (SEO).
<link rel="stylesheet" ...>: Menghubungkan file HTML dengan file desain eksternal (CSS).
<style>: Tempat menuliskan aturan desain (CSS) langsung di dalam file HTML.
<body>: Tempat semua konten yang ingin diperlihatkan kepada pengunjung diletakkan.
<h1> s/d <h6>: Tag judul bertingkat. <h1> adalah yang paling penting (paling besar) dan <h6> yang terkecil.
<p>: Digunakan untuk menulis teks dalam bentuk paragraf.
<a>: Membuat link (hyperlink) yang bisa diklik untuk berpindah halaman atau situs. Atribut href berisi alamat tujuannya.
<img>: Menampilkan gambar. Atribut src adalah sumber file gambar, dan alt adalah teks pengganti jika gambar gagal dimuat.
<div>: Wadah atau kontainer "kosong" untuk mengelompokkan elemen agar mudah diatur tata letaknya (biasanya menggunakan kelas/class).
<ul>: Membuat daftar poin (simbol titik) yang tidak berurutan (Unordered List).
<ol>: Membuat daftar berurutan menggunakan angka atau huruf (Ordered List).
<li>: Item di dalam daftar. Digunakan baik di dalam <ul> maupun <ol>.
